# Introduction to AI - Laboratory

This repository contains lab exercises for the ENSIA S2 Introduction to AI course.

## Contents

| File | Topic |
|------|-------|
| Lab2_Intro_Python.ipynb | Python Fundamentals |
| Lab3__Agent_Environement.ipynb | Agent-Environment Interaction |
| LAB4____Agents.ipynb | AI Agents |
| Lab5____Search__Part_I.ipynb | Search Algorithms (Part I) |
| Lab6__informed_Search_EMPTY.ipynb | Informed Search (A*, Greedy Best-First) |
| Lab7_Local_Search.ipynb | Local Search (Hill Climbing) |
| LAB8_V1_Empty.ipynb | Advanced Local Search (Stochastic, First Choice, Random Restart, Simulated Annealing) |
| LAB9_V1_Empty.ipynb | Genetic Algorithms (Selection, Crossover, Mutation) |

## Requirements

- Python 3.8+
- Jupyter Notebook / JupyterLab
- Required packages listed in each lab notebook

## Usage

Open any notebook with Jupyter:

```bash
jupyter notebook
```

Then navigate to the desired lab file.
